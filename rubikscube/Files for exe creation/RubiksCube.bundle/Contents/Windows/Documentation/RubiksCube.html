Description

This app simulates 3x3 Rubik's cube. The cube has different colors on each face. Each face can be rotated in clockwise or anti-clockwise direction. The user solves the puzzle by bringing consistent colors on all the faces.

Following are the features:

User can interactively solve the Rubik's cube.

User can scramble the configuration of the Rubik's cube to create a puzzle.

User can see the solution of the cube solver.

General Usage Instructions

Go to environments tab.

Click on Rubik's Cube button.

Click Solve button to play with cube interactively. Drag the face in the desired direction to rotate the face clockwise or anti-clockwise.

Click Scramble button to allow app to generate a puzzle by scrambling the colors of the faces.

Click Play button to solve the cube automatically and see all the moves as animation.

Click Next to stop the ongoing animation and see each step one by one.

Commands

Ribbon/Toolbar Icon Command Command Description

Solve

Allows user to play the Rubik's cube interactively. User can select any face and drag the face in desired direction to rotate it clockwise or anti-clockwise.

Scramble

Allows user to randomize the colors on the faces to create a scrambled cube.

Play

Solves the current configuration of the cube and allows user to see all the moves as animation.

Next

Solves the cube if it is unsolved and plays each move on the click.

Installation/Uninstallation

The installer that ran when you downloaded this app/plug-in from Autodesk Exchange Store has already installed the app/plug-in.

You can unload the plugin without uninstalling it by unchecking the "Loaded/Unloaded" check box associated with the plugin in the Inventor Add-In Manager.

You may have to restart Inventor to completely unload the application.

To remove the plugin completely, Right Mouse Button click and uninstall the application using the Exchange App Manager tool, located on the Ribbons Tools tab > Options Panel.

To uninstall this plug-in, exit the Autodesk product if you are currently running it, simply rerun the installer, and select the "Uninstall" button.

OR, you can uninstall it from 'Control Panel\Programs\Programs and Features' (Windows 7/8/8.1) or Add/Remove Programs (Windows XP), just as you would uninstall any other application from your system.
